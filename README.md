# BlueOrion QMS

This is the BlueOrion Quality Management System (QMS) project.  
It includes a Node.js backend (server.js) and a frontend for document control, welfare monitoring, and admin dashboards.

## How to run locally
1. Install dependencies: `npm install`
2. Start the server: `node server.js`
3. Open your browser at http://localhost:3003

## Features
- Admin dashboard
- Document management
- Welfare monitoring
- User authentication
